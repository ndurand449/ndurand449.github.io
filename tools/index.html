My Tools

This website gathers information from the Internet that I use regularly for decision-making purposes.

Currency Tool: EUR/USD

This tool uses a combination of actual exchange rates year-to-date and analysts' forecasts for the remainder of the year to calculate estimated average exchange rates for the EUR/USD pair for the current year.

I originally created this tool for the Python for Everybody Capstone course. Since May 2022, I have been running this tool locally on my computer every month.

For this portfolio, I made adjustments to the Python programs to generate output files displayed on this page and Javascript data files used to create a Google Chart graph.

Treasury Tool: Treasuries

This tool presents selected data from the U.S. Treasury. Interest rates for treasury bills, notes, and bonds are shown in different Google Charts over certain time periods.

I originally created this tool as an Excel spreadsheet for weekly usage.

For this page, I developed a Python program to transform a csv file that I download from the U.S. Treasury website into Javascript data files that are then displayed on this page in a Google Chart table and Google Chart graphs.
